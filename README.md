# WALL·E Terminal Animation

A simple Python animation of WALL·E pacing across a junkyard scene.

![demo](demo.gif)

## Requirements
- Python 3.10+
- A terminal supporting inline images (Kitty, WezTerm, Konsole, etc.)
- Linux or macOS (tested on Arch Linux)

## Installation
```bash
git clone https://github.com/JGGHalfhide/wall-e.git
cd wall-e
pip install -r requirements.txt
python walle.py
